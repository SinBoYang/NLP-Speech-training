# AI Speech Training System

本專案是一個基於自然語言處理（NLP）與語音技術的 AI 演說訓練系統。  
使用者可以透過錄音或上傳音訊檔案，將語音轉換成逐字稿，並透過後端 AI 分析演講內容，提供演說改善建議、文字改寫、翻譯與語音合成功能。

系統主要整合了 Speech-to-Text（STT）、Large Language Model（LLM）、Text-to-Speech（TTS）、翻譯服務、文本分析與情緒分析，協助使用者提升演講的結構、用詞、節奏與說服力。

---

## 專案目標

本系統的目標是協助使用者練習與改善演說表現。  
使用者可以輸入或上傳自己的演講內容，系統會分析逐字稿中的問題，例如冗詞過多、詞彙重複、結構不清楚、情緒張力不足等，並提供具體修改方向。

此外，系統也能參考具影響力演說者的風格，例如川普、馬丁路德金恩博士、黃仁勳等，將原始演講稿改寫成不同風格的版本，幫助使用者學習不同的演說表達方式。

---

## 主要功能

### 1. 語音轉文字 Speech-to-Text

使用者可以上傳音訊檔案，系統會透過 Azure Speech-to-Text 將語音內容轉換成文字逐字稿。

支援格式包含：

- wav
- mp3
- ogg
- flac
- m4a
- wma
